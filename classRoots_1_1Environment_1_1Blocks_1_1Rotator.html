Detailed Description

A rotator exhibits a continuous rotation. Attach this script to a tile block to make it spin.

By parenting multiple rotators to the same object, it's possible to create interesting wheel designs by using the startDepth of each rotator to control the distance between them. However, since each block modifies the transform of the parent object, they cannot rotate in opposing directions. However, that can be accomplished by simply using two sets of wheels and carefully choosing position, direction, and speed.
